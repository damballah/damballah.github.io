Damballah le savoir doit se partager

Le but de ce multi site est de partager des informations d’ordres informatiques, médicales, scientifiques, historiques, paléontologiques… Que sais-je encore… Jugées utiles par son administrateur… Hmmm moi.

Pourquoi “multi site”?

Parce qu’il est aussi bien partagé sur le web dit de “surface” que sur le web dit “profond”, ainsi que sur le réseau P2P Bitorrent et sur le web permanent utilisant le protocol IPFS.

Ainsi, Il est hébergé :
  • Chez un hébergeur officiel
  • Sur Github
  • Sur le réseau DAT
  • Sur le réseau ZeroNet
  • Sur le réseau IPFS
  • Sur le DeepWeb (ou… Darknet comme vous préférez)
Internet, c’est la boîte de pandore…
